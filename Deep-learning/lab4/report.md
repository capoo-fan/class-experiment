# 基于 lab4.ipynb 分步策略的双能谱图像校正实践

## 摘要

本实验针对双能谱医疗成像处理中产生的成对基质图像 `(f, g)` 进行深度学习校正。实验使用 `phantom_input_train.npy` 与 `phantom_target_train.npy` 构造监督学习数据，将含噪模拟基质图像映射到无噪真实基质图像，并将训练后的模型分别应用于 25 组模拟测试图像与 25 组真实医疗测试图像。本文直接在 `lab4.ipynb` 中完成代码改写，沿用原 notebook 的分步实验策略：导入库、设置超参数、加载数据、观察样例、构建卷积神经网络、训练模型、绘制损失曲线、输出预测结果。模型采用双通道残差 U-Net，在 macOS M4 环境下可通过 PyTorch MPS 后端进行 GPU 加速，最终输出 50 张预测结果图片。

## 关键词

双能谱图像；图像去噪；卷积神经网络；残差学习；U-Net；

## 1. 图像去噪的背景及意义

双能谱成像设备可以采集两组能量谱下的医学图像数据，并通过基质分解方法得到成对的基质图像 `(f, g)`。由于实际采集过程会受到量子噪声、探测器误差、散射和重建算法局限等因素影响，得到的基质图像往往包含噪声、条纹或局部伪影。如果直接使用这类图像进行后续观察和分析，可能影响组织结构判读与定量测量。

图像去噪的目标是在尽量保留真实结构边缘和灰度分布的前提下降低噪声。本实验将双能谱图像校正近似为监督式图像去噪任务：输入含噪图像，输出对应的无噪模拟图像，通过神经网络学习噪声模式与校正映射，使测试图像更接近理想图像。

## 2. 深度学习图像去噪方法

传统去噪方法通常依赖人工设计滤波器，例如均值滤波、高斯滤波、中值滤波和非局部均值等，但这类方法容易在抑制噪声的同时模糊边缘。深度学习方法能够利用大量样本自动学习图像局部纹理、边缘结构和噪声分布，常见模型包括 DnCNN、去噪自编码器、U-Net 和生成对抗网络等。

卷积神经网络适合处理图像数据，卷积层能够提取局部空间特征，编码器-解码器结构能够结合不同尺度的上下文信息，残差学习则可以让模型重点学习输入图像与目标图像之间的噪声差异，而不是从零生成完整图像。因此，本实验在原 `lab4.ipynb` 的 CNN 实验框架上，将分类网络改写为面向双通道图像校正的残差 U-Net。

## 3. 本文使用的方法

模型输入张量形状为 `(N, 2, 256, 256)`，两个通道分别对应 `f` 与 `g`。网络采用编码器-解码器结构，并通过跳跃连接融合浅层空间细节与深层语义特征。模型最后输出双通道残差，预测结果为：

```text
corrected_image = noisy_input - predicted_residual
```

训练时先将输入和目标按固定尺度 `1.5` 归一化到较稳定的数值区间，并对极端噪声值进行截断。损失函数由 L1 损失、MSE 损失和梯度差异损失组成，其中 L1 损失提高整体灰度恢复能力，MSE 损失约束像素误差，梯度损失用于保持边缘结构。训练集按 9:1 划分为训练集和验证集，并使用随机翻转、旋转进行数据增强。

## 6. 总结（困难或调试）

本实验的主要难点有三点。第一，训练数据为 `float64` 且体积较大，若一次性载入内存会占用过多资源，因此 notebook 中使用 `numpy.load(..., mmap_mode="r")` 进行内存映射读取，并在 batch 内转换为 `float32`。第二，输入图像存在少量噪声值，直接训练可能导致损失不稳定，因此使用固定尺度归一化与截断来提高训练稳定性。第三，真实医疗测试图像与模拟训练图像分布不同，模型在真实数据上的泛化比模拟测试集更困难，因此采用数据增强和梯度损失改善结构保持能力。

通过本实验可以看到，沿用原 `lab4.ipynb` 的分步 CNN 实验策略，可以较清晰地完成双能谱图像校正任务。卷积神经网络能够学习双能谱基质图像中的噪声特征，并对模拟测试图像取得较好的去噪效果。若进一步提高结果质量，可以增加训练轮数、扩大模型通道数，或在真实医疗图像上加入自监督适配方法。

## 附录（代码）

核心模型结构如下：

```python
class Net(nn.Module):
    def __init__(self, base_channels=16):
        super().__init__()
        c1, c2, c3 = base_channels, base_channels * 2, base_channels * 4
        self.enc1 = ConvBlock(2, c1)
        self.enc2 = ConvBlock(c1, c2)
        self.bottleneck = ConvBlock(c2, c3)
        self.pool = nn.MaxPool2d(2)
        self.up2 = nn.ConvTranspose2d(c3, c2, kernel_size=2, stride=2)
        self.dec2 = ConvBlock(c2 + c2, c2)
        self.up1 = nn.ConvTranspose2d(c2, c1, kernel_size=2, stride=2)
        self.dec1 = ConvBlock(c1 + c1, c1)
        self.residual_head = nn.Conv2d(c1, 2, kernel_size=1)

    def forward(self, x):
        e1 = self.enc1(x)
        e2 = self.enc2(self.pool(e1))
        b = self.bottleneck(self.pool(e2))
        d2 = self.up2(b)
        d2 = self.dec2(torch.cat([d2, e2], dim=1))
        d1 = self.up1(d2)
        d1 = self.dec1(torch.cat([d1, e1], dim=1))
        residual = self.residual_head(d1)
        return x - residual
```
